SEMANA 01 - ESTRUCTURA DEL SISTEMA OPERATIVO LINUX

En esta actividad representé la estructura básica del sistema de archivos de Linux utilizando Minecraft.

La construcción está basada en el directorio raíz ( / ), que es donde inicia toda la organización del sistema.

Se representaron los siguientes directorios:

/ → Directorio principal del sistema.
/boot → Archivos necesarios para el arranque.
/home → Archivos personales de los usuarios.
/etc → Configuraciones del sistema.
/bin → Comandos esenciales.
/usr → Programas instalados.
/var → Archivos que cambian constantemente, como registros.

En el video explico la función de cada uno y su importancia dentro del sistema operativo.
