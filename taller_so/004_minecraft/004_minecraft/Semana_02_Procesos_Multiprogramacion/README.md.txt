SEMANA 02 - PROCESOS Y MULTIPROGRAMACIÓN

En esta actividad representé cómo funcionan los procesos dentro de un sistema operativo.

Se construyeron diferentes zonas que simulan procesos ejecutándose al mismo tiempo, además de una zona que representa el control del sistema.

Se incluyeron:

- Procesos activos funcionando simultáneamente.
- Zona de administración (CPU / Sistema Operativo).
- Zona donde un proceso finaliza.

En el video explico qué es un proceso y cómo el sistema operativo permite que varios procesos se ejecuten al mismo tiempo (multiprogramación).
