SEMANA 03 - VIRTUALIZACIÓN

En esta actividad representé el concepto de virtualización utilizando una estructura por niveles.

La construcción incluye:

- Hardware físico (base).
- Sistema operativo host.
- Hipervisor.
- Máquinas virtuales.
- VPS.

En el video explico cómo el hipervisor permite crear máquinas virtuales dentro de un mismo equipo físico y cómo estas funcionan de manera independiente.
